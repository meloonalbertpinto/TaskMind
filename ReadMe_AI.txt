Prothon AI - Gemini 2.0 Integration
======================================

Features Added:
- Extract Summary, Key Points, Action Items, and Key Decisions from live transcriptions.
- Interactive Q&A: Ask questions about the meeting via the sidebar chat.
- Powered by Google Gemini 2.0 Flash (High availability and reliability).
- Built-in session storage in the `sessions/` directory.

How to use:
1. Ensure your Gemini API Key is in `analyzer.py`.
2. Run the application: `python app.py`
3. Start a transcription session.
4. Click "Insights" or use the "Ask about the meeting" chat.

Model Used:
- gemini-2.0-flash (Updated for speed and reliability)
